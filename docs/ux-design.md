# UX / UI Considerations

This document captures the primary UX patterns and design considerations for the core mobile experience.

## Layout
- Tabbed/bottom navigation: feeds, saved items, settings
- Feed list view with folders/collections and quick actions
- Article list with filtering toolbar (unread, starred, tags)
- Full-screen reader view with toolbar (mark read, star, share) and swipe gestures

## Navigation
- Gesture navigation (swipe left/right to move between items, pull-to-refresh)
- Tinder-style swipes in Reader: left (previous), right (next), up (full article mode)
- Quick actions (toggle read, open in browser, share link)
- Auto-scrolling and infinite list support

## Accessibility
- Screen reader support (TalkBack on Android, VoiceOver on iOS)
- High contrast / dark mode
- Large touch targets for gestures
